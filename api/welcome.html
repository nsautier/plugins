General notes

Anytime 'Paginated' is mentionned as true, This means, that the request needs the X-Range Header to be specified with the start and end included indexes specifying the range that API user needs to fetch from given collection.

The X-Lang header can be filled with language codes, that aren't ISO codes, but simple 'conventional' 2 letter codes which we will moderate one day. Mostly used as of 2015 are 'fr', 'en' and 'es'.

You're welcome on http://plugins.glpi-project.com/#/submit to read about the XML Glpi Plugin Description File specification, and here see how you can specify information in multiple languages (tags, description, ...)

This header change some special properties of resources fetched, and will use translation if possible on specific fields.

Limitation of usage

Scopes

NameDescription
pluginsView all known plugins
pluginGet card of a single plugin
plugin:starGrants right to note a plugin
plugin:submitGrants right to post a new plugin
tagsView all known attributed tags
tagView card of a single tags
authorsView all known contributors

Endpoints served by this API

Contributors

  • Contributor list

    URL: /author Method: GET Description: List of GLPI known contributors Paginated: true (answers 206 Partial Content)

    Example usage (HTTP Session) 

                   
GET /api/author HTTP/1.1
Host: plugins.glpi-project.com
X-Lang: en
X-Range: 0-14

    Example usage (cURL call) 

                   
curl -X GET -H 'X-Lang: en' -H 'X-Range: 0-14' http://plugins.glpi-project.com/api/author

  • Contributor card

    URL: /author/<id> Method: GET Description:Description of a GLPI contributor

    Parameters

    • Name: id Description: ID of the GLPI contributor to look at
    Paginated: false

    Example usage (HTTP Session) 

                   
GET /api/author/3 HTTP/1.1
Host: plugins.glpi-project.com
X-Lang: en

    Example usage (cURL call) 

                   
curl -X GET -H 'X-Lang: en' http://plugins.glpi-project.com/api/author/3

  • Contributor plugin list

    • Name: id Description: ID of the GLPI contributor to look at
    Paginated: true

    Example usage (HTTP Session)

Host: plugins.glpi-project.com

Plugins

  • Search

    URL: /search Description: The collection of all GLPI known plugins matching the query_string Method: POST

    Parameters (JSON Payload)

    • Name: query_string Description: Search string to query
    Paginated: true (answers 206 Partial Content)

    Example usage (HTTP Session) 

                   
POST /api/search HTTP/1.1
Host: plugins.glpi-project.com
Content-Type: application/json
X-Lang: en
X-Range: 0-14

{"query_string": "additional"}

    Example usage (cURL call) 

                   
curl -X POST -H 'X-Lang: en' -H 'X-Range: 0-14' -H 'Content-Type: application/json' -d '{"query_string": "additional"}' http://plugins.glpi-project.com/api/search
